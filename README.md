## Development

Run it locally

```zsh
git clone git@github.com:stargately/beancount-mobile.git
cd beancount-mobile
npm install
npm run start
```

Scripts

- `npm run test`: test the whole project
- `npm run lint`: run the linter
- `npm run schema`: generate Apollo GraphQL schema
